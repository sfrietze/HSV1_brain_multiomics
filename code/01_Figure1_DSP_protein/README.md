# Figure 1: DSP protein and ROI quantification

This folder contains scripts for Figure 1 DSP protein analysis and Supplementary Figure 1 normalization controls.

Expected inputs:
- ROI-level HSV-1 antigen and Iba1 area measurements
- GeoMx DSP normalized protein abundance matrix
- ROI metadata including condition, antigen status, mouse ID, and ROI ID

Expected outputs:
- Figure 1C-D ROI quantification plots
- Figure 1E DSP protein differential abundance plot
- Figure 1F DSP protein heatmap
- Figure 1G selected protein plots
- Supplementary Figure 1 DSP normalization controls
