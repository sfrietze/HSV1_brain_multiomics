# Intranasal HSV-1 Infection Drives Region-Specific Interferon-Dominant Microglial Remodeling

Code and data supporting the analyses presented in **Frietze et al.**, *PLOS Pathogens* (under revision).


## Repository organization

```
code/       Analysis scripts organized by figure
data/       Processed data used for analyses
outputs/    Representative outputs generated by the workflows
raw_data/   Large reference files and intermediate data
```

Figure-specific scripts are located in:

- `code/01_Figure1_DSP_protein/`
- `code/02_Figure2_multiome_annotation/`
- `code/03_Figure3_DEG_pathways/`
- `code/04_Figure4_SCENIC_footprinting/`
- `code/05_Figure5_spatial_transcriptomics/`

## Data availability

Raw sequencing data have been deposited in GEO:

- **GSE324144** – Single-cell multiome profiling of mouse brain following HSV-1 infection
- **GSE324176** – Spatial transcriptomic profiling of HSV-1-infected mouse brain using the NanoString GeoMx Whole Transcriptome Atlas

Large analysis files required for reproducing selected figures are available through Zenodo.

## Running the analyses

Each figure directory contains the scripts required to reproduce that figure. Analyses were performed using R and Python.
